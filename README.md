# QTyper - Тренажер слепой печати

## Основные возможности
* **Игровой процесс**: характеризуется случайным выбором текста из внутреннего хранилища приложения, разделенного по языкам, а также автоматически обрабатываемым в соответствии выбранным уровнем сложности;
* **Виртуальная клавиатура**: один из главных помощников тренажера. Подсвечивает следующий нужный для ввода символ, а также подсказывает, каким пальцем его следует нажимать;
* **Визуализация рук**: следующий главный помощник, отображает левую и правую руку с подсвеченными пальцами для правильного нажатия на клавишу;
* **Статистика**: сохранение истории попыток: скорость, точность, затраченное время. Статистика позволит объективно оценить ваши навыки владение слепой печати или же процесс овладения;
* **Настройки**: все необходимое для комфортной игры находится здесь. Регулировка громкости звука и музыки, выбор языка интерфейса, переключение режим окна позволят сосредоточиться на игре;
* **Пиксельный стиль**: достояние тренажера, уникальный ретро-дизайн с анимированными кнопками и частицами на фоне, поможет комфортному времяпрепровождению и испытанию некоторой ностальгии.

## Инструкция по запуску
### Для пользователей
Зайти в раздел **Releases** на странице проекта. Скачать архив с проектом, выбрав один из трех для своей системы, распаковать в любую папку (кроме системных) и запустить игру.

### Для разработчиков
1. Если есть желание покапаться в исходном коде и также изменить его под свои нужны, можно просто скачать архив с проектом нажав на зеленую кнопку `Code` и в открывшемся меню кнопку `Download zip`

2. Если вы умеете пользоваться консолью, то выполните следующие действия:
* прописать или скопировать команду в командную строку
```
git clone https://github.com/AleshaHD/QTyperRpg.git
```
* для запуска игры из консоли нужно перейти в папку куда вы скопировали проект и прописать
```
.\main.py
```

  ### Пример того как скачать проект и запустить его через консоль на Windows
* Открываем консоль, например через комбинацию клавишь `win + s` или же просто открыв меню поиска и прописав `cmd`, после чего появится для выбора **Командная строка**. Или же второй способ, комбинация клавишь `win + r`, откроется меню **Выполнить** и в нем прописать команду `cmd`, после чего также откроется командная строка.
* Прописываем команду
```bash
git clone https://github.com/AleshaHD/QTyperRpg.git C:\QTyper
```
* Далее переходим в эту папку
```
cd C:\QTyper
```
* После чего прописываем команду для запуска программы
```
.\main.py
```
