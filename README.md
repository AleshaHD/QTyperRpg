# QTyper - Тренажер слепой печати
***
* игровой процесс - характеризуется случайным выбором текста из внутреннего хранилища приложения, разделенного по языкам, а также автоматически обрабатываемым в соответствии выбранным уровнем сложности;
* виртуальная клавиатура - один из главных помощников тренажера. Подсвечивает следующий нужный для ввода символ, а также подсказывает, каким пальцем его следует нажимать;
* визуализация рук - следующий главный помощник, отображает левую и правую руку с подсвеченными пальцами для правильного нажатия на клавишу;
* статистика - сохранение истории попыток: скорость (WPM), точность (%), затраченное время. Статистика позволит объективно оценить ваши навыки владение слепой печати или же процесс овладения;
* настройки - все необходимое для комфортной игры находится здесь. Регулировка громкости звука и музыки, выбор языка интерфейса, переключение режим окна позволят сосредоточиться на игре;
* пиксельный стиль - достояние тренажера, уникальный ретро-дизайн с анимированными кнопками и частицами на фоне, поможет комфортному времяпрепровождению и испытанию некоторой ностальгии.
